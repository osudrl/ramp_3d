Grounding Vision and Language to 3D Masks for Long-Horizon Box Rearrangement

Reactive Action Mask Planner (RAMP-3D)

Ashish Malik, Caleb Lowe, Aayam Shrestha, Stefan Lee, Fuxin Li, Alan Fern
Paper Code

Abstract

We study long-horizon planning in 3D environments from under-specified natural-language goals using only visual observations, focusing on multi-step 3D box rearrangement tasks.

RAMP-3D formulates planning as sequential prediction of paired 3D masks: a which-object mask indicating what to pick and a which-target-region mask specifying where to place it. The system processes RGB-D observations and natural-language task specifications to reactively generate multi-step pick-and-place actions.

Across 11 task variants in warehouse-style environments with 1-30 boxes, RAMP-3D achieves a 79.5% long-horizon success rate and substantially outperforms 2D VLM-based baselines.

Key Points

  • Planning directly in 3D mask space
  • No symbolic planning layer
  • 11 language-conditioned task variants
  • 96.8% one-step validity
  • 79.5% long-horizon success
